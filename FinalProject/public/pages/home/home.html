Welcome to Private Shelf

Private Shelf is a book catalog application. The aim is to provide a friendly user experience that anyone can use to help create a private cataloging service let it be in public libraries or private home shelves. As a catalog service, the application will allow the user to store book information sending information about the book's name, author, publish date, description, and page. The application will communicate with the firebase server uploading the book information and allow users to search for books within the server using a search query.
